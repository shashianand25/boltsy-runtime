# Workflow Patterns

## Feature Workflow

Task intake -> context aggregation -> route model -> spawn agent -> validate -> review -> notify.

## Bug Fix Workflow

Reproduce -> isolate cause -> patch -> regression test -> validate -> notify.

## Review Workflow

Load diff -> load policies -> classify risk -> review with criteria -> return structured decision.

## Documentation Workflow

Load context -> draft concise doc -> verify commands and paths -> update memory if the doc changes operating behavior.
